# 🏏 IPL Data Analysis – Tableau Dashboard

An interactive Tableau dashboard built to explore and visualize Indian Premier League (IPL) match data across seasons. The project uncovers key patterns in team performance, player statistics, and match outcomes through dynamic, filterable visuals.

🔗 **[View Live Dashboard on Tableau Public](https://public.tableau.com/app/profile/aryan.gadkar/viz/IPL_16838941742450/Analysis)**

---

## 📊 Overview

The Indian Premier League (IPL) is one of the most-watched T20 cricket tournaments in the world. This project uses historical IPL data to build a comprehensive visual analysis covering match-level and player-level statistics.

---

## 🔍 Key Insights Covered

- **Match Statistics** – Season-wise match results, venues, and win margins
- **Team Performance** – Win/loss records, toss impact, and home vs. away analysis
- **Player Performance** – Top run-scorers (Orange Cap), leading wicket-takers (Purple Cap), and boundary hitters
- **Toss Analysis** – How toss decisions influenced match outcomes across seasons
- **Venue Analysis** – Ground-specific trends and team advantages

---

## 🛠️ Tools & Technologies

- **Tableau Public** – Data visualization and dashboard design
- **Data Source** – IPL dataset (matches and ball-by-ball deliveries)

---

## 📁 Dataset

- `matches.csv` – Match-level data for each IPL game
- `deliveries.csv` – Ball-by-ball data for all matches
